# Astro Tech Blog

Read this blog at [Tech-Blog](https://tech-blog-31y.pages.dev)

A simple Astro blog scaffold with:

- React component development
- MDX and Markdown article support
- Left-side nested navigation
- Right-side ads panel
- Top header and bottom footer
- SEO-friendly metadata
- Cloudflare Pages static deployment compatibility

## Local commands

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Deployment

This site is configured for static export and Cloudflare Pages.
- Build command: `npm run build`
- Publish directory: `dist`
- `robots.txt` is available at `/robots.txt`
- RSS feed is available at `/rss.xml`
- Sitemap is generated automatically at `/sitemap.xml`

> Replace the placeholder site URL in `astro.config.mts` with your real domain before publishing.
