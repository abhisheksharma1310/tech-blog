## Print Styles

CSS can create styles specifically for printed pages using the `print` media type.

### Common Print Techniques

```css
@media print {
  /* Hide non-essential elements */
  nav, .sidebar, .advertisement { display: none; }

  /* Make links visible */
  a[href]::after {
    content: " (" attr(href) ")";
  }

  /* Page breaks */
  h2 { page-break-before: always; }

  /* Ensure backgrounds appear */
  * {
    -webkit-print-color-adjust: exact;
    print-color-adjust: exact;
  }
}
```

### @page Rule

Control the printed page dimensions and margins:

```css
@page {
  size: A4;
  margin: 2cm;
}

@page :first {
  margin-top: 4cm;
}
```

### Print-Specific Properties

| Property | What It Does |
|---|---|
| `page-break-before` | Break before an element (`always`, `avoid`) |
| `page-break-after` | Break after an element |
| `page-break-inside` | Avoid breaks inside an element |
| `orphans` | Min lines at bottom of page |
| `widows` | Min lines at top of page |

Always test your print styles — what looks good on screen may not translate well to paper.
