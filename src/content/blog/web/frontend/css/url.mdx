## url()

The `url()` function is used to reference external resources like images, fonts, and SVGs.

### Where url() Is Used

| Property | Example |
|---|---|
| `background-image` | `background-image: url("bg.png");` |
| `list-style-image` | `list-style-image: url("bullet.svg");` |
| `border-image` | `border-image: url("border.png");` |
| `cursor` | `cursor: url("custom.cur"), auto;` |
| `@font-face` | `src: url("font.woff2");` |

### Relative vs Absolute URLs

```css
/* Relative — resolves relative to the CSS file */
background: url("images/bg.png");

/* Absolute — full URL */
background: url("https://cdn.example.com/bg.png");
```

Always use quotes around the URL for clarity, though they are technically optional.
